Game Tips

Tank: I'm not trying to kill, I'm trying to control

Orisa

Push in waves - peek, bait, retreat

Fortify -> Spin -> Fire -> Jav -> Reset

Role Identity: You’re the anchor. You control space and protect your team by absorbing key cooldowns.

Main Tempo: Push in waves — 2–3 seconds of pressure, then reset behind cover. Don’t overextend.

Other Notes:

  • Use corners – peek, bait, retreat.
  • Your gun isn’t poke — it’s pressure.
  • Don’t spray all game.

Core Cooldown Rhythm: Fortify → Spin (as Fortify ends) → Pressure fire → Javelin → WAIT → repeat.

Most Important Notes:

  • Trade Fortify for one enemy cooldown to gain an advantage.
  • Anchor between your supports and DPS, not far ahead.

Junker Queen

Push, pause, punish, retreat

Shout -> Swing wide out from cover -> Knife -> Axe -> Shotgun -> Reset

Role Identity: You’re pressure + sustain. Win fights through anti-heals, bleeds, and careful trades. You’re strongest when controlling small angles or chokes.

Main Tempo: Push → Pause → Punish → Retreat. Don’t overcommit; fight in rhythm.

Other Notes: Corner fighting only; avoid open fields. Start axe before knife pull.

Core Cooldown Rhythm: Shout → Swing wide out from cover → Knife / Knife Pull → Axe → Shotgun/melee finish. Reset behind cover.

Most Important Notes:

  • Never push without Shout.
  • Win fights with tempo, not raw brawl.

Winston

Jump in -> Bubble -> Zap -> Jump Out -> Reset -> Repeat.

Second dive only after cooldown

Role Identity: You're the fight starter. Dive, create chaos, break setups, open space for DPS to follow through.

Main Tempo: Tag a target → leave → re-dive.

Other Notes:

  • Mark one high-value target per fight.
  • Always ensure: 1) You're in healer LOS, 2) You have bubble ready, 3) Know your exit, 4) Your team can follow up.

Core Cooldown Rhythm: Jump in -> Bubble -> Zap -> Jump Out -> Reset -> Repeat. Second dive occurs after Jump cooldown resets.

Most Important Notes:

  • Commit only if jump hits.
  • Use Bubble to separate enemies.

D.Va

Short dive → peel → reposition

Boosters In → Microburst → Melee → Defense Matrix → Boosters Out → repeat.

Role Identity: Flex diver and peel machine. Deny key abilities, chase isolated targets, and protect supports.

Main Tempo: Short dive → peel → reposition. Constant mobility is key.

Other Notes: Stagger enemies to deny resets and peel efficiently.

Core Cooldown Rhythm: Boosters In → Microburst → Melee → Defense Matrix → Boosters Out → repeat.

Most Important Notes:

  • Use Matrix to deny high-value cooldowns.
  • Reposition constantly; angles > HP.

Roadhog

Hook -> pressure -> reset

Hook → Shotgun → Melee → Follow-up → Hide → repeat

Role Identity: Pick tank / off-angle anchor. Off angle tank, not a main choke brawler. Your goal is one elimination per cycle, turning 5v5 into 5v4.

Main Tempo: Hook -> pressure -> reset

Other Notes: If you're in open space you're feed, if you're in a choke or behind a corner, you're a threat. Ultimate for picks or zoning, not random spray. Always step behind cover when breathing.

Core Cooldown Rhythm: Hook → Shotgun → Melee → Follow-up → Hide → repeat.

Most Important Notes:

  • Hook first, then commit.
  • Play safe if hook is down.

Mauga

Walk until it’s time to kill — then dash

Overrun In → Burn (Crit Rage) → Cardio sustain → Back off → repeat

Role Identity: Juggernaut brawler. Win fights by taking space, burning targets, and using Overrun + Cardio to dominate.

Main Tempo: Walk forward → force cooldowns → Overrun engage → back off when Cardio ends.

Other Notes: Focus squishies or low-mobility tanks. Walk until it’s time to kill — then dash.

Core Cooldown Rhythm: Overrun In → Burn (Crit Rage) → Cardio sustain → Back off → repeat.

Most Important Notes:

  • Use Overrun to punish and force cooldowns.
  • Always corner yourself; open ground = death.

Zarya

Use your bubble proactively

Self Bubble → push → Projected Bubble → beam burn → WAIT → repeat

Role Identity: Frontline bully. Win slow brawls by trading bubbles effectively and building high charge.

Main Tempo: Bubble → push → reset → repeat.

Other Notes: High charge (70+) = dictate fights. Use your bubble proactively, just as the enemy commits.

Core Cooldown Rhythm: Self Bubble → push → Projected Bubble → beam burn → WAIT → repeat.

Most Important Notes:

  • Trade bubbles for maximum value.
  • High charge = control fight tempo.

Moira

Fade for escape or reposition only

Survivability > greedy plays

Role Identity: Sustain + damage support. Your value comes from sustain + consistent DPS, not flashy kills. Heal allies while dealing damage to charge Fade/Coalescence.

Main Tempo: Mid-range → heal → poke → retreat → repeat.

Other Notes:

  • Fade for survival/escape/rotations only - NOT panic.
  • Damage to charge and pressure.
  • Use Fade to escape or reposition, never offensively unless safe.
  • Focus the same target as the one your heal target is focusing.
  • Look past the tank and do damage on support, then DPS, then tank.

Core Cooldown Rhythm: Heal → Damage orb/beam → Fade → Coalescence → repeat.

Most Important Notes:

  • Heal efficiently, prioritize survival over greedy plays.
  • Use Fade wisely

Zenyatta

Cover -> Charge right click -> Peek & volley damage -> Hide back behind cover

Role Identity: High-value poke healer. Deny healing, apply Discord, sustain with Harmony.

Main Tempo: Poke → Discord → reposition → repeat.

Other Notes: Discord = fight-decider. Harmony = pre-emptive sustain. Stay safe; Zen dies quickly. If in doubt, heal orb on tank, discord on tank.

Core Cooldown Rhythm: Harmony → Discord → Orb poke → Transcendence → repeat.

Most Important Notes:

  • Keep Discord on high-value targets
  • Stay safe while healing

Kiriko

Heal → Kitsune → Kunai → Suzu → repeat.

Role Identity: Mobility + poke healer. Keep allies alive and pressure flankers.

Main Tempo: Heal → teleport → Kunai poke → Protection Suzu → repeat.

Other Notes: Teleport = reposition/save. Suzu = reactive save. Track ally positions.

Core Cooldown Rhythm: Heal → Kitsune → Kunai → Suzu → repeat.

Most Important Notes:

  • Use Suzu to save key allies
  • Teleport to high-impact positions/saves

Ana

Heal → Biotic Grenade → Sleep Dart → Nano / poke → reset → repeat.

Role Identity: Long-range poke healer. Heal allies, deny enemy abilities, swing fights with utility.

Main Tempo: Hold angle → heal → poke → anti-heal → retreat → repeat.

Other Notes: Sleep = pick tool. Biotic = deny healing/secure kills. Nano = fight-winning combo. Position for line-of-sight.

Core Cooldown Rhythm: Heal → Biotic Grenade → Sleep Dart → Nano / poke → reset → repeat.

Most Important Notes:

  • Sleep/biotic on key threats
  • Stay alive in safe angles

Mercy

Heal → Damage Boost → Guardian Angel → Resurrect → Valkyrie → repeat.

Role Identity: Primary sustain + clutch healer. Keep key allies alive and enable dives.

Main Tempo: Heal priority ally → damage boost → Guardian Angel reposition → repeat.

Other Notes: Prioritize high-impact allies. Resurrect = fight-turner. Guardian Angel = reposition/escape. Valkyrie = mobility + enhanced healing.

Core Cooldown Rhythm: Heal → Damage Boost → Guardian Angel → Resurrect → Valkyrie → repeat.

Most Important Notes:

  • Keep priority allies alive
  • Use Guardian Angel to stay mobile and safe

General tank

“I’m not trying to kill — I’m trying to control.”

“Peek → bait something → hide → peek during their downtime.” You fight in bursts, not continuous exposure.

Don’t dive or push alone — push with purpose and support line-of-sight.

If your supports can’t see you, you’re not creating space — you’re feeding.

Think of yourself as the engine of the team: start, stop, and pace the fight.

If in doub't, slow down.

General support

Survival

Don't be greedy for kills

Use off angles.

Warm up

Be deliberate

Use bottom half of crosshair

Focus on the target - like the blade of a fan

  1. Tracking the target without shooting
  2. Deliberate shots – aim for 100% accuracy, use a metronome
  3. 180 flicks to target acquisition
  4. Always intend for head shots
  5. Focus on the target not just look at it (eg. blade on fan, not fan itself)
  6. Hero specific aim training (eg. Ana sleep dart, Orisa jav, Junker Queen jagged blade)
  7. Hero specific cooldown combo training (eg. Junker Queen: Shout → Swing wide out from cover → Knife / Knife Pull → Axe → Shotgun/melee finish)
  8. 50–100 kills
  9. A-D strafe aiming, crosshair at head height, no mouse movement
Cassidy

Hold angle → poke → punish → rotate → repeat.

Take safe angles for 2-taps.

Role Identity: Mid-range duelist + pick maker. Secure kills with stickies, 2-taps, and Fan.

Main Tempo: Hold angle → poke → punish → rotate → repeat.

Other Notes: Play mid-range. Sticky = punish commit. Roll → reload → Fan = burst damage. Use off-angles.

Core Cooldown Rhythm: Take angle → poke → Sticky on commit → Fan + roll cancel → reset.

Most Important Notes:

  • Take safe angles for 2-taps
  • Sticky only on committed targets

Ashe

Hold angle → poke → dynamite → punish → reposition → repeat.

Stay high ground and rotate often.

Role Identity: Mid–long-range pressure and pick DPS. Control space with dynamite and punish overextensions.

Main Tempo: Hold angle → poke → dynamite → punish → reposition → repeat.

Other Notes: Dynamite softens teams. Coach Gun = spacing + escape. High ground = safety + value. B.O.B. controls space.

Core Cooldown Rhythm: Scoped shots → Dynamite → Unscoped shots → Coach Gun → reset → repeat.

Most Important Notes:

  • Dynamite pressure wins fights early
  • Stay high ground and rotate often

Reaper

TP → Walk → Burst → Wraith → Reset.

Use Wraith to survive.

Role Identity: Close-range assassin. Delete tanks, supports, isolated targets with flanks.

Main Tempo: Flank → burst → Wraith out → repeat.

Other Notes: Don’t fight main lines. Wraith = survival + reload. Coordinate with team.

Core Cooldown Rhythm: TP → Walk → Burst → Wraith → Reset.

Most Important Notes:

  • Always flank; avoid main sightlines
  • Use Wraith to survive

Symmetra

Set up → hold close angle → punish entries → push when charged.

Fight in tight spaces

Role Identity: Area control + brawl DPS. Hold chokes, force enemies through turrets + beam.

Main Tempo: Set up → hold close angle → punish entries → push when charged.

Other Notes: Turrets = zone, protect supports. Teleporter = mobility/escape. Fight in tight spaces.

Core Cooldown Rhythm: Turrets → Beam poke → Orb pressure → Push while charged.

Most Important Notes:

  • Fight in tight spaces
  • Place turrets before engagement

Tracer

Always keep a guaranteed escape (Blink or Recall).

Save a Blink, don’t overstay.

Role Identity: Assassin + disruptor. Pressure backlines, force cooldowns, finish low targets.

Main Tempo: Poke → force cooldowns → burst → Recall out → rotate → repeat.

Other Notes: Commit only with an exit. Save a Blink, don’t overstay. Pulse targets with no mobility. Play off-angles.

Core Cooldown Rhythm: Blink → burst → Blink/strafe → Recall → reposition → repeat.

Most Important Notes:

  • Always keep a guaranteed escape (Blink or Recall)
  • Constant pressure on supports wins fights
