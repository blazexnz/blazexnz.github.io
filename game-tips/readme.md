# Game Tips Guide 🎮

A mobile-friendly, interactive web app for gamers to access hero-specific tips, cooldown rhythms, and warmup exercises. Quickly filter heroes by role or purpose and get bite-sized, actionable advice—perfect for quick reference before or during gameplay.

## Features

- **Role-Based Filtering:** Instantly filter tips by **Tank**, **DPS**, **Support**, **Warmup**, or **General** strategies  
- **Rank Filtering (NEW):** Use the **Rank dropdown** to filter tips by rank (e.g. **Bronze**, **Gold**, **Gold to Masters**)  
- **Role-Based Rank Defaults (NEW):** When you select a role, the rank filter can automatically default to a preferred rank for that role (e.g. Tank → Bronze, Support → Gold)  
- **Rank Metadata Per Tile (NEW):** Each hero tile can include `data-ranks="..."` so the app knows which ranks that tip applies to  
- **Hero Tiles:** Each hero has a dedicated tile with:
  - Key highlights and tempo tips
  - Role identity and responsibilities
  - Core cooldown rhythms and sequences
  - Most important notes for gameplay  
- **Warmup Guidance:** Step-by-step exercises for aiming and tracking  
- **Key Banner:** Sticky banner for overarching tips or reminders  
- **Mobile Optimized:** Responsive design, large touch-friendly tiles, works on smartphones and tablets  
- **Interactive Filters:** Tap a button to show only relevant heroes or exercises  
- **Collapsible Notes:** Click the banner to dismiss it for a clean view  

## How to Use

1. Open the app in your browser (mobile or desktop).  
2. Use the **filter buttons** at the top to select a role:
   - All, Tank, DPS, Support, Warmup, General  
3. (NEW) Use the **Rank dropdown** to filter tips by rank:
   - All ranks, Bronze, Gold, Gold to Masters  
4. Scroll through the **hero tiles** to find tips for that character.  
   - **Highlights:** Key combos, rhythms, and tempo  
   - **Role Identity:** Understand the hero’s purpose  
   - **Core Cooldown Rhythm:** Suggested ability sequences  
   - **Most Important Notes:** Critical reminders for play  
5. Dismiss the **key banner** by tapping it if you want a distraction-free view.  
6. Use the app before matches or during warmups for a quick refresher.
